# Unity Endless Runner --- HW1

## Описание проекта
ФИО: Албогачиев Алим Ахмедович (БПИ249) 

Проект представляет собой прототип игры-раннера.

Игрок автоматически движется вперёд по трёхполосной трассе, может
переключаться между линиями и выполнять двойной прыжок. По мере
прохождения скорость увеличивается, появляются препятствия и бонусы. При
потере всего здоровья происходит перезапуск сцены.

# Игровая механика
**Управление**:
-   A / D - переключение между линиями
-   Space - прыжок (поддерживается двойной прыжок)

**Основная логика игры**:
1.  Игрок автоматически движется вперёд.
2.  Скорость постепенно увеличивается со временем.
3.  Спавнер генерирует препятствия и бонусы (монеты и аптечки)
4.  При столкновении с препятствием уменьшается здоровье
5.  При достижении 0 HP сцена перезапускается


# Алгоритм работы
**Игровой цикл**:
-   PlayerController в FixedUpdate():
    -   управляет движением вперёд через Rigidbody.velocity
    -   выполняет переключение линий
    -   обрабатывает прыжки
-   GameManager:
    -   хранит текущий счёт
    -   управляет множителем скорости
    -   отвечает за перезапуск сцены
-   ObstacleSpawner:
    -   генерирует препятствия и бонусы 
    -   удаляет объекты позади игрока
-   GroundTileManager:
    -   реализует бесконечную землю путём переиспользования тайлов


**Реализация систем**:\
Движение игрока:
-   Используется Rigidbody
-   Движение вперёд - через изменение velocity.z
-   Переключение линий - через MovePosition
-   Прыжок - через AddForce

Система здоровья:
-   Хранится в PlayerController
-   Метод TakeDamage() уменьшает HP
-   При HP = 0 вызывается Die() и GameManager перезапускает сцену.

Бонусы:
-   Реализованы через ScriptableObject (PickupData)
-   Каждый prefab бонуса содержит ссылку на данные эффекта
-   При OnTriggerEnter начисляются очки или здоровье, после чего объект
    уничтожается

UI:
-   Реализован через Canvas + TextMeshPro
-   HUD отображает HP, Score, Speed


# Точки входа
**Главная сцена:** Assets/Scenes/Main.unity \
**Основной управляющий скрипт:** GameManager.cs - Отвечает за скорость, счёт и перезапуск сцены. \
**Основной игровой скрипт:** PlayerController.cs - Реализует движение, прыжки, здоровье и обработку столкновений.
