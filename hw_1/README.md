# Endless Runner (hw_1)

Прототип бесконечного раннера на Unity: игрок бежит вперёд по трёхполосной дороге, уворачивается от препятствий и собирает бонусы.

## Игровой процесс
- Постоянное движение вперёд; скорость плавно растёт от x1.0 до ~x2.2 за 120 секунд (см. `GameManager`).
- Три полосы движения со сменой дорожки и двойным прыжком (`PlayerController`).
- Столкновение с препятствием отнимает здоровье, при 0 HP происходит рестарт сцены.
- Подбираемые бонусы: добавление очков или лечение (`PickupData`).
- Очки растут со временем и за бонусы; HUD показывает HP, Score и текущий множитель скорости.

## Управление (клавиатура)
- `A` / `←` — смещение влево.
- `D` / `→` — смещение вправо.
- `Space` — прыжок (до 2 подряд, если настроено `maxJumps`).

## Требования
- Unity **6000.3.10f1** (из `ProjectSettings/ProjectVersion.txt`).
- TextMeshPro установлен по умолчанию в проекте.

## Структура папок
- `Assets/Scripts/Core` — менеджмент игры, камера, бесконечные плитки, HUD.
- `Assets/Scripts/Player` — движение и здоровье игрока, конфиг.
- `Assets/Scripts/Obstacles` — препятствия, бонусы, спавн логика.
- `Assets/Scenes` — основная сцена `SampleScene`.
